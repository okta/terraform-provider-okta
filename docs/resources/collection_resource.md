# Resource: okta_collection_resource

Manages a resource within an Okta Governance Collection. This resource allows you to add applications with specific entitlements to a collection.

~> **Note:** This resource requires Okta Governance (Okta Identity Governance - OIG) to be enabled in your Okta organization.

## Example Usage

```terraform
resource "okta_collection" "slack_permissions" {
  name        = "Slack Bot Permissions"
  description = "Entitlements for in-house Slack bot"
}

resource "okta_collection_resource" "slack_app" {
  collection_id = okta_collection.slack_permissions.id
  resource_orn  = "orn:okta:idp:${var.org_id}:apps:salesforce:${var.app_id}"
  
  entitlements {
    id = okta_entitlement.feature_a.id
    values {
      id = okta_entitlement.feature_a.values[0].id
    }
  }
  
  entitlements {
    id = okta_entitlement.feature_b.id
    values {
      id = okta_entitlement.feature_b.values[0].id
    }
    values {
      id = okta_entitlement.feature_b.values[1].id
    }
  }
}
```
